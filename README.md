# EntregaFinal-Gatica

SmokingCute E-commerce - Backend API:
SmokingCute es una API RESTful desarrollada en Java Spring Boot para un e-commerce especializado en la venta de ceniceros de diversas variedades y personalizados. Este backend proporciona todas las funcionalidades necesarias para gestionar productos, pedidos, usuarios y el carrito de compras.

Características Principales:
Gestión de Productos
° Listar todos los productos disponibles
° Búsqueda de productos por nombre o descripción
° CRUD completo de productos
° Categorización de productos
° Control de stock y alertas de stock bajo
° Productos personalizables
Gestión de Pedidos:
° Creación de pedidos con validación de stock
° Historial de pedidos por usuario
° Seguimiento de estado de pedidos (Pendiente, Confirmado, Enviado, Entregado, Cancelado)
° Actualización automática de stock al confirmar pedidos
° Cálculo automático de totales
Sistema de Carrito:
° Agregar/eliminar productos del carrito
° Actualizar cantidades
° Persistencia del carrito
Administración:
° Gestión de categorías
° Reportes de stock bajo
 Gestión de usuarios:
 Arquitectura Tecnológica
Backend
Java 11 - Lenguaje principal
Spring Boot 2.7.0 - Framework principal
Spring Data JPA - Persistencia de datos
MySQL 8.0 - Base de datos relacional
Maven - Gestión de dependencias
Características Técnicas:
° API RESTful con endpoints documentados
° Validación de datos con Spring Validation
° Manejo de excepciones personalizadas
° Configuración CORS para desarrollo frontend
° Documentación automática con OpenAPI/Swagger
