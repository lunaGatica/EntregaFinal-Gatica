# EntregaFinal-Gatica
SMOKINGCUTE - E-commerce de Ceniceros de Diseño
° Descripción:
SmokingCute es una aplicación web de e-commerce especializada en la venta de ceniceros de diseño. El sistema incluye backend desarrollado en Java Spring Boot con MySQL y frontend en React, ofreciendo una experiencia de compra personalizada.

° Características Principales
Backend (Java Spring Boot)
✅ API RESTful completa
✅ Gestión completa de productos (ceniceros)
✅ Sistema de personalización de productos
✅ Gestión de pedidos y usuarios
✅ Autenticación y autorización
✅ Validaciones y manejo de excepciones
✅ Integración con base de datos MySQL
Frontend (React)
✅ Catálogo interactivo de ceniceros
✅ Sistema de filtros avanzados
✅ Carrito de compras
✅ Personalizador de productos
✅ Panel de administración

✅ Diseño responsive y moderno

✅ Gestión de pedidos en tiempo real
